Rows & Columns

A container contains a collection of items that can be stacked horizontally or vertically (no Tetris). An item is a simple, dumb unit that takes all the space it was given to grow in.

Columns

Fits as many equally wide columns of at least 220px (3 columns on the grid) as possible in the space it is given. Distributes the number of items intelligently across columns. Naturally responsive: can be put in any context.

Rows

A row is a list of 2 to 4 variable height units that are stacked by nature, and horizontally aligned on wider breakpoints. An item can vary in width: the higher the coefficient, the more space it will take compared to its siblings.

The highest item defines the height of a row.

Code

<div class="u-row">
    <div class="u-row__item"></div>
    <div class="u-row__item"></div>
    …
</div>
